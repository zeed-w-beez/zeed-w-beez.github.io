很久没更新过了。

最近突然很想写点东西。

原因很简单，自己之前遇到过的问题，没有记录下来；现在又遇到了，但是已经忘记了。

还是要多写写，给自己看也好。

准备自己的一些东西：技术虽然发展很快，但是真正实用的技术目前还是有边界的。

  1. 前端
    • 涉及到的不是很多，但是基本的 es6, react, redux, angularjs, typescript 也是要知道的。
    • webpack 是个好工具。
    • npm 除了有点慢，其他还行吧。
  2. 后端是主攻吧
    • 登录的就 Oauth2.0
    • 编程语言，能用c#就用吧，毕竟是这个出身。而且 dotnet core 1.0 已经出来了，用起来还不错。
    • 框架什么的，没有最好，只有最适合。
    • 自己写的Framework？呵呵，我很懒，够用就好了。不够了再自己撸~
  3. 数据存储
    • 看情况，自己的小东西，最喜欢用Sqlite。
    • 现在开始尝试 gogis 涉及到 Postgresql ，也挺不错的。
    • 开源界用户量最庞大的是 MySql 吧。可以根据需要接。
    • 其实微软的售后是最好的。目前接触到的情况来看。
    • Redis, mongodb 看需要使用。个人比较喜欢Redis，社区比较好，用的多。
  4. 大数据，目前还用不到，但是了解下还是有好处的。
    • Google file system 超大文件存储, Taobao File System 海量小文件存储 …
    • hadoop = mapreduce + 什么来着… 分块存储，离线计算，汇总统计 是解决大数据分析的基本原则吧。
    • habase bigtable …
  5. Linux
    • 为什么越来越喜欢 Linux 了，因为开源可用的东西太多了。
  6. Tensorflow
    • 没事玩玩挺好的。
  7. 其他
    • git
    • vs
    • docker
