tensorflow

是个好东西，自己也接触下。

Linux 上都好办，就不说了。

Windows 上有个 Python 的坑，记录下：

Tensorflow 对 Windows 支持，目前是只有 Python3 .
想要做 Windows 上跑，必须要Python 3，目前 Python 最新的版本是 3.6.x，但是 Tensorflow 才支持到 3.5，所以要下载 3.5.x 版本的Python 。
再用 pip install tensorflow 才行。

等以后都支持了，就不用Care这个了。
