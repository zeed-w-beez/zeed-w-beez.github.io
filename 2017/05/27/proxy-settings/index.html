proxy settings

2017-05-27 边边角角

代理是个好东西，但是… 不想多说什么

Windows

有些代理需要输入用户名和密码。
在Windows下是默认是自动集成的，不需要自己输入。

但是如果想给某些软件做配置，没有密码还不行。特别是给虚拟机里的系统配置代理的时候。

这个时候就多绕一步。

Fiddler 本身就是个代理软件，打开后会自动在本机8888起个端口，接手全部的请求。

在 Fiddler 的 Rules 里面发现了一个 Automatically Authenticate 选项，可以自动做权限验证，避免了 407 Proxy Authentication Required 错误。

就可以直接给需要的软件配置 Fiddler 8888 的代理。

还可以当做是一层保险，不开Fiddler 的时候，虚拟机是不能访问网络的。

大部分软件的代理配置都可以百度到，这里记录些不常用的：
