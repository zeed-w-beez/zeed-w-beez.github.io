yum install mysql

2017-05-06 术业有专攻

最近在折腾Linux，少不了要安装一些乱七八糟的东西。
用的是Centos，所以都是 yum 这个包管理直接安装。
安装小东西很简单方便，比如 vim, git 这些。

但是大一点的软件，比如 mysql ，就会有些麻烦了。

还是有很多办法解决的。

不过现在我开始用 Postgresql 了。

原因么：没有好坏，只有适合不适合。

安装了 gogs ，本来只想用 sqlite 的，不过还需要自己编译源代码，改配置啥的，懒的搞。
Postgresql 可以直接用 yum 安装，也没有啥配置，就用了。
