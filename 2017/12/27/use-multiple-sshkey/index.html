use multiple sshkey

2017-12-27 边边角角

日常的工作生活中，我们可能会遇到不止一个 github/gitlab 服务器。虽然 ssh key 可以避免每次和服务器交互的时候都要输入密码，但是如果我想区分工作和生活的 ssh key 又该怎么做呢？其实只需要一个简单的 config 文件就可以了。

1.检查是否已经存在 sshkey

如果可以看到类似 id_rsa.pub id_dsa.pub 文件，可以跳过第二步创建。
