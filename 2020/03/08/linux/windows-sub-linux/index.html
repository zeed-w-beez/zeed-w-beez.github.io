windows sub linux

2020-03-08 术业有专攻

新版本的 Windows 10 自带了 Windows Sub Linux 功能。可以更方便的使用 linux 系统了。

安装

首先要在 程序和功能 里启用 WSL 功能。

然后去 windows store 搜索 liunx 并下载自己喜欢的发行版。

整个操作都比较简单。

问题

root 用户的密码

  1. 刚安装完之后，可以直接使用 sudo passwd root 设置 root 账号的密码。
  2. 如果是忘记密码，可以用 CMD wsl -u root 直接以 root 身份进入系统，想做什么都可以。
  3. 如果你安装了多个 wsl，可以使用 wsl -d Debian -u root 指定某个 wsl。

切换默认登录账号

  1. 使用 ubuntu.exe /? 查看某个子系统的帮助命令
  2. 设置默认账号的命令 ubuntu config --default-user root
